### 2️⃣ 生词本显示全部 1460 个单词
是的，生词本默认列出**全部单词**，支持：

| 功能 | 说明 |
|------|------|
| **搜索** | 输入英文/中文关键词模糊匹配 |
| **全部** 标签 | 显示所有 1460 个单词（最多 100 个，防止渲染卡顿） |
| **掌握** 标签 | 只显示你标记为 ✓ 的 |
| **模糊** 标签 | 只显示你标记为 ⭐ 的 |
| **未标记** 标签 | 只显示还没碰过的 |

如果觉得 1460 个太多，可以用搜索或者筛选标签快速定位。
