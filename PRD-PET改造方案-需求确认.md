# PET 备考工具 — 需求分析与改造方案

> 版本：v0.2（需求确认阶段）
> 日期：2026-06-23
> 状态：**需求已确认，待进入开发**

---

## 一、产品定位

**一句话**：一个让 8 岁孩子每天愿意打开、能坚持到 PET 通过的专属学习工具。

不是"背单词 App"，而是"弟弟的 PET 闯关游戏"。

---

## 二、目标用户画像

| 维度 | 描述 |
|------|------|
| 年龄 | 8 岁，三年级 |
| 英语基础 | 零基础 ~ 少量学校英语 |
| 注意力 | 每次有效学习 10-15 分钟 |
| 使用场景 | 每天放学后、周末，家长监督或自主 |
| 动机 | 通过 PET 考试（目标驱动，但需要游戏化包装） |
| 已有能力 | 已会简单使用平板/手机 |

**关键约束**：
- 单次学习量不宜超过 5-8 项
- 界面文字要大、色彩要活泼、交互反馈要即时
- 复习频次需要高于成人，因为遗忘曲线对儿童更陡

## 三、内容资源分析

### 3.1 三套资源的定位

| 资源 | 形式 | 数量 | 定位 | 可交互性 |
|------|------|------|------|---------|
| PET 高频词汇表 | 结构化文本（词性+释义） | ~2000 词 | **核心学习材料** | 高 |
| PET 必背短语 | 结构化文本（短语+释义） | 327 条 | **核心学习材料** | 高 |
| KET/PET 语法 PDF | 扫描图片（139+151页） | 290 页 | **辅助参考** | 低 |

### 3.2 词汇与短语的关系

**已确认：全部纳入，每天 30 词 + 10 短语**

两份材料来自不同来源，约 **80% 的短语核心词出现在词汇表中**，约 **20% 独立**。

**关联策略**：先学含该词的短语，再学独立短语。

```
学习单元 = 关联词 + 关联短语 + 当日独立短语
```

举例：
```
Day 1:
  go          → 配套短语: go on / go out / go ahead
  come        → 配套短语: come in / come back / come out
  break       → 配套短语: break down / break in
  独立短语 × 4
```

**已确认：每天 30 词 + 10 短语，约 65 天完成一轮**（2000÷30 ≈ 67 + 327÷10 ≈ 33，重叠部分综合考虑）。

### 3.3 语法难度递进

**关键发现**：两本语法 PDF 均为**纯扫描图像**，无嵌入式文本层，Python/PyMuPDF/fitz 均无法提取任何文字。转为 Word 也无法提取（扫描页本质上是图片）。

**解决方案**：根据剑桥官方 KET→PET 语法大纲，编写**结构化语法学习指南**（纯文本，可交互），PDF 作为翻页参考配套：

| 阶段 | 词汇难度 | 语法重点 | PDF 参考 | 建议时长 |
|------|---------|---------|---------|---------|
| **阶段 1：基础构建** | 日常高频词（动物/食物/家庭/颜色/学校） | 名词复数、一般现在时、be/ have | KET PDF 第 1-40 页 | 前 15 天 |
| **阶段 2：句子扩展** | 动作/描述/方位/时间 | 现在进行时、情态动词(can/must)、比较级 | KET PDF 第 41-80 页 | 第 16-30 天 |
| **阶段 3：过去时与完成** | 抽象词/情感/学术类 | 一般过去时、现在完成时、被动语态 | KET 81-139 + PET 1-30 | 第 31-50 天 |
| **阶段 4：PET 进阶** | 全量词汇复习 | 条件句(if)、定语从句、间接引语 | PET 第 31-90 页 | 第 51-65 天 |
| **阶段 5：PET 冲刺** | 全量短语 + 造句 | 综合语法巩固 + 真题演练 | PET 第 91-151 页 | 第 65 天后 |

> 语法指南将实现为**可点击的知识卡片**，每个语法点包含：规则说明 + 例句 + 小练习。PDF 作为"查阅 PDF 原文"的附加入口。

## 四、功能模块设计

### 4.1 整体结构（底部导航）

| Tab | 名称 | 功能 |
|-----|------|------|
| 🏠 | **首页** | 今日学习计划 + 闯关进度条 + 打卡 |
| 📚 | **学习** | 每日 30 词 + 10 短语学习（主流程） |
| 🔄 | **复习** | 间隔重复 + 模糊词回炉 + 听写 |
| 📖 | **词库** | 全部 PET 词/短语浏览、搜索、筛选 |
| ⚙️ | **设置** | 进度重置、考试目标 |

> 删除了：日历（并入首页统计）、AI 对话、用户切换、哥哥账号

### 4.2 学习流程

```
每天打开首页
    ↓
今日任务：30 新词 + 10 短语（分 5-6 组，每组学完有小反馈）
    ↓
点"开始今日学习"
    ↓
第 1 组：6 个词 → 词卡模式
  英文展示 → 猜释义 → 点击翻转 → 自评 ○/⭐/✓
    ↓
第 1 组配套短语：含这 6 个词的短语展示
    ↓
第 2-5 组：重复以上模式
    ↓
当日独立短语展示（无法匹配到今日词汇的短语）
    ↓
进度更新 + 撒花 🎉
```

### 4.3 复习策略

| 机制 | 说明 |
|------|------|
| 间隔重复 | 1-2-4-7-15 天递增（儿童遗忘更快） |
| 模糊回炉 | 复习日自动聚合 ⭐ 词 |
| 听写 | 选 5 个词，中文 → 拼写英文 |
| 连续未打卡 | 不惩罚，仅提醒"弟弟今天还没学哦" |

### 4.4 游戏化

| 元素 | 方案 |
|------|------|
| 等级 | Lv.1 新手 → Lv.10 PET 考生 |
| 进度环 | "PET 备考进度"（基于已学词数/总数） |
| 连续打卡 | 🐣 → 🐤 → 🐔 → 🦅 动物进化 |
| 完成动画 | 每日任务完成后撒花 |

## 六、数据准备

语法 PDF 是扫描件，Python 无法直接提取文字。我将采用以下方式：

1. **PET 词汇表** — 用 Python 解析 PDF 提取全部 ~2000 词（已验证可提取）
2. **PET 短语** — 用 Python 从 DOCX 提取全部 327 条（已验证可提取）
3. **语法指南** — 我根据剑桥 KET/PET 体系**手写完整语法知识卡片**，绑定 PDF 页面为参考

> 语法 PDF 不需要你转换格式。扫描版的字转出来也是乱码。我直接按照 KET→PET 的语法体系编写交互式知识卡片，PDF 作为原始的"翻书查看"入口。
