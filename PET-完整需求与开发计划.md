## 三、功能结构

### 3.1 底部导航（5 项）

| Tab | 名称 | 功能 |
|-----|------|------|
| 🏠 | **首页** | 今日学习计划、闯关进度条、打卡入口 |
| 📚 | **学习** | 每日 30 词 + 短语的学习卡片流（主流程） |
| 🔄 | **复习** | 模糊词回炉 + 间隔重复 + 听写 |
| 📖 | **词库** | 全部 PET 词/短语浏览、搜索、筛选 |
| ⚙️ | **设置** | 进度重置、每日目标设置 |

### 3.2 学习流程（核心）

```
每天打开 → 首页显示今日任务
   ↓
今日任务卡片：Day N/90 | 已学 XX 词
   ↓
点"开始今日学习"
   ↓
分组学习（每 6 词 + 2 短语为一组，共 5 组）
  每组内：
    词卡模式：英文 → 猜释义 → 翻转 → 自评 ○/⭐/✓
    关联短语：展示含该词的短语（标记 📗或📘）
   ↓
全部完成后 → 撒花动画 + 打卡按钮
```

### 3.3 复习策略

| 机制 | 说明 |
|------|------|
| 间隔重复 | 1-2-4-7-15 天递增 |
| 模糊回炉 | 复习页聚合所有 ⭐ 词 |
| 听写 | 两种模式可选：**当日**（今天全部 30 词）或 **之前日**（随机从前面的词中抽 30 个）。显示中文→拼写英文，错误自动标记为 ⭐ |
| 连续打卡 | 🐣 → 🐤 → 🐔 → 🦅 动物进化 |

### 3.4 游戏化

| 元素 | 方案 |
|------|------|
| 等级 | Lv.1 新手 → Lv.10 PET 考生 |
| 进度环 | 基于已学词数 / 2679 的环形进度 |
| 连续天数 | 动物图标升级 |
| 完成反馈 | 每日结束后撒花 🎉 |

### 3.5 语法功能

两本语法 PDF 是扫描版，不作为交互内容。改为：
- **语法知识卡片**：每个语法点一张卡片（规则 + 例句 + 小练习）
- 按 5 个阶段（KET 入门 → PET 冲刺）分布到 90 天中
- 卡片中附带"参考 PDF 第 XX 页"的翻看入口

## 八、iPad 上的语音与安装方案

### 8.1 网页语音（TTS）在 iPad 上的可行性

**结论：能用，但有局限。**

| 场景 | 效果 |
|------|------|
| 网页朗读英文单词（speechSynthesis） | ✅ 支持，iOS Safari 从 iOS 7 起支持 Web Speech API |
| 需要用户点一下才发声 | ⚠️ iOS 限制：第一次发声必须是用户触发的（点击按钮） |
| 音量/自动播放 | ⚠️ iOS 静音开关下无声音，需要确认 iPad 音量已开 |
| 离线朗读 | ❌ 依赖系统内置语音包，断网时部分语音可能不可用 |
| 语音质量 | 一般。系统内置的英文语音（Samantha/Alex）质量尚可，但远不如 App 级 AVSpeechSynthesizer |

**旧版已经在用 `window.speechSynthesis`，直接复用即可。**

### 8.2 安装成 App 的三种方案对比

| 方案 | 代价 | 安装体验 | TTS 可靠性 | 推荐度 |
|------|------|---------|-----------|--------|
| **PWA（推荐）** | 加 2 个文件（manifest.json + service worker），**几小时** | 添加至主屏幕，全屏无地址栏，像原生 App | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ 首选 |
| **Capacitor 壳** | 安装 Node 包 + 配置，打包后上架 App Store，**2-3 天** | 真正原生壳，可提交 App Store | ⭐⭐⭐⭐ | ⭐⭐⭐ 备选 |
| **Native App** | 全部重写 iOS 版，**数周至数月** | 完全原生 | ⭐⭐⭐⭐⭐ | ❌ 不推荐 |

### 8.3 PWA 方案详解

PWA = 现在的网页 + 两个配置文件，**不改一行业务代码**：

```
vocab-app/
  ├─ src/ （现有代码不变）
  ├─ public/
  │   ├─ manifest.json     ← 新增：图标、名称、主题色
  │   └─ sw.js             ← 新增：离线缓存（可选）
  └─ index.html            ← 加一行 <link> 引用 manifest
```

**添加后，弟弟在 iPad Safari 上：**
1. 打开 `scdq.online`
2. 点分享按钮 → 「添加至主屏幕」→ 桌面出现 App 图标
3. 以后直接点图标打开，**全屏运行，没有地址栏和标签页**
4. 离线也能打开已学内容
